# Amp Sims Plugins

Guitar and bass amplifier simulators for instrument processing.

## Overview

Amp simulation plugins model the behavior of guitar and bass amplifiers, from clean to heavily distorted.

**Plugin Count**: 13  
**Recommended**: 2  
**Latest**: 13

## Key Plugins

### Recommended

## Common Use Cases

- **Guitar tones**
- **Bass processing**
- **DI recording**
- **Reamping**


## All Plugins in This Category

### Cabs *Latest*

### CrickBass *Latest*
